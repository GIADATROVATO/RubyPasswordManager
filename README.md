Password Manager Ruby 
Un semplice password manager scritto in Ruby che salva e carica credenziali in modo sicuro usando AES-256-CBC.  

--> Funzionalità
- Inserimento di credenziali (sito, nome utente, password)
- Visualizzazione delle credenziali salvate
- Cifratura dei dati con master password
- Salvataggio e caricamento sicuro da file

--> Requisiti
- Ruby >= 2.7
- Librerie:
  - openssl
  - json
  - io/console (per input nascosto della password)
